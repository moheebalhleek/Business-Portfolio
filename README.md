# Somethin’s Brewin’ - Strategic Marketing & Data Analysis

## 1. The Strategic Challenge
The downtown coffee market is saturated with commercial chains. Our objective was to transition from "habitual buying" to "value-based organic consumption" by implementing a digital-first strategy to engage modern consumers.

## 2. Execution Strategy (The 5A Journey)
We utilized the 5A framework to guide the customer journey, ensuring every touchpoint was optimized for mobile engagement.

| Stage | Objective | Execution Tool |
|-------|-----------|----------------|
| **Awareness** | Penetrate the market | Teaser Content |
| **Appeal** | Build quality trust | Cinematic Visuals |
| **Ask** | Drive curiosity | Targeted Questions |
| **Act** | Ensure conversion | Free Trial App Offer |
| **Advocate** | Create loyalty | Referral Programs |

## 3. Performance & Impact (Data-Driven Validation)
The following dashboard represents the real-world application of our strategy. By utilizing SQL-based analytics, we tracked the campaign’s impact against our defined SMART goals over the first 90 days.

| Metric | Goal | Actual Result | Status |
| :--- | :--- | :--- | :--- |
| **New Customers** | 1,000 | 1,120 | ✅ Exceeded |
| **App Adoption** | 60% | 65% | ✅ Exceeded |
| **Delivery Orders** | 20% | 18% | ⚠️ On Track |
| **CAC (Avg. Cost)** | $5.00 | $4.20 | ✅ Optimized |

### Key Insights
* **Data Methodology:** Aggregated real-time transactional data using SQL queries to monitor conversion rates.
* **Result:** The alignment of the 'Free Trial' campaign with our 'Mobile-first' strategy directly resulted in exceeding customer acquisition targets.
